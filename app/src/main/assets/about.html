You Control the Ads.

About:

Ad Wheel gives you control over the ads. Here you can play and tweak with many of the publicly available settings Google uses to render advertisements. There is only one page with direct access to be able to search for particular categories or set up a random spin wheel with any categories you wish.

Features:

  • View mainstream mobile advertisements on demand by controlling and configurting a random topic wheel
  • Control and request particular ad topics directly by using keywords via the search tab.
  • Set your personal settings to see how they affect which ads are rendered.
  • View and replay your viewing history by topics and time.

Disclaimer:

Ad Wheel attempts to find the most appropriate ad matching the user's search query. Keywords and topics are provided directly into the ad request engine. There is no guarantee that an ad will come back matching the user's original query - advertisements shown will vary depending on availability.
